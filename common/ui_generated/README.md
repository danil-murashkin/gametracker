## `common/ui_generated/`

Сюда кладутся `ui.c`, `ui.h`, `ui_events.*`, `ui_logic.*`, **сгенерированные** в `editor/` (вкладка **Code**).

### Симулятор (браузер)

Проверка без железа — вкладка **Simulator** в [`../../editor/`](../../editor/): компилируется и запускается тот же C (Emscripten + LVGL).

### Прошивка

Скопируйте экспорт в `common/ui/` (или сюда) и пересоберите [`../../firmware/`](../../firmware/).

### Примечания

- Эти файлы **перезаписываются** при каждом экспорте из редактора.
- В репозитории лежит минимальный пример (таймер + счётчик) как «smoke test».
